The AlgoryalAPI is the core of all Algoryal projects. The API is accessible by anyone but is designed for internal use. We use the API to access descriptions, titles, lists, social links, and resources. The API is the central information container for the company.

How We Use The API

We have developed tools for accessing data from the API. We have a Python module and a few tools for accessing the API via JS and Jekyll. All of our tools are available on GitHub and are licensed under the GNU GPL v.3.0.

How The API Works

The API consists of a Firebase Real-time Database containing all information. We access the data by simply doing requests to the database’s JSON files. It works like this:

  • We store some basic and general info about the company in the database in a bucket called: “Info”.
  • We can access the data by going to this URL: https://algoryalapi.firebaseio.com/Info.json.
  • The URL consists of a firebase-url, a sub-folder/bucket and a .json extension.
